# Starfield cli



#### Flags

```
   --background value, --bg value, -b value                       background to use (default: 0)
   --stars value, --star value                                    number of stars to draw (default: 500)
   --width value                                                  Resolution width (default: 800)
   --height value                                                 Resolution height (default: 600)
   --swidth value, --starfield_width value, --starfieldw value    Starfield width (default: 1200)
   --sheight value, --starfield_height value, --starfieldh value  Starfield height (default: 1200)
   --help, -h                                                     show help (default: false)                              show help (default: false)
---
```

### Example

```
go run cmd/starfield/main.go -width=1000 -height=800 -bg=1
```
